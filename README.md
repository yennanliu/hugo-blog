# hugo-blog

## Setup

```bash
brew install hugo
```

## Run

```bash

# hugo new site quickstart
# cd quickstart
# git init
# git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
# echo "theme = 'ananke'" >> hugo.toml
# hugo server


# run app
hugo server

# http://localhost:1313
```
